Me chamo Vinicius, sou um brasileiro de 19 anos nascido em Novo Hamburgo e morador de São Leopoldo, descendente de imigrantes alemães.

Sou formado em Técnico em Desenvolvimento de Sistemas pela instituição Senac de São Leopoldo, o curso técnico teve duração de um ano e meio e me introduziu de forma abrangente no mercado de TI, digo que o curso me introduziu de forma abrangente pois já havia feito antes três cursos de curta duração também oferecidos pela instituição Senac de São Leopoldo que me introduziram de forma inicial na área, sendo um de introdução a lógica de programação, um de introdução aos testes de software e um de introdução à linguagem Python.

Durante o curso trabalhei e continuo trabalhando bastante com VS_Code, um editor de código bastante usado no desenvolvimento de softwares. Trabalhei com linguagens como HTML e CSS para desenvolver sites estáticos sem muitos segredos, mas utilizei Javascript para sites com funcionalidades mais dinâmicas, o website que desenvolvi durante minhas aulas de Front-end que ainda estou desenvolvendo é meu site-portfólio, esse que você está vendo. Além de websites estáticos ou funcionais, desenvolvi sistemas de API que faziam a integração do banco de dados com os sites web, principalmente relacionados a cadastro e Login de usuários no site, o banco de dados utilizado para todo o curso foi o famoso mySQL.
